# Smart Poultry IoT

Système intelligent low-cost pour surveiller et automatiser un poulailler (température, humidité, qualité d'air NH₃/CO₂, poussière, niveau d'eau, porte automatique, ventilation).

## Fonctionnalités principales (Sprint 1)

- Authentification sécurisée (éleveur & admin)
- Gestion de profil utilisateur
- Liste et ajout de poulaillers

## Technologies

- **Embarqué** : ESP32 + PlatformIO
- **Backend** : Node.js + Express + MongoDB
- **Mobile** : React Native + Expo
- **Communication** : MQTT (Mosquitto)
- **Authentification** : JWT
